# Relatório sobre Otimização Contínua e Aprendizado de Máquina

Este repositório contém o código-fonte e os recursos para o relatório sobre Otimização Contínua e Aprendizado de Máquina.

## Estrutura do Repositório

- `sec*/`: diretórios nesse formato contém as seções (ou capítulos) do relatório.
- `figures/`: estão as imagens e figuras utilizadas no relatório.
- `references.bib`: Arquivo de referências em formato BibTeX.

## Como Compilar
`make build`
